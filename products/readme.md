# Base de datos
## Estandar de nombres
**__[tl_mod_tablename]__**
* **t**  = Tipo de objeto
* **md** = modulo, debe tener 2 caracteres
* **nombre** = nombre relacionado a los datos que contiene

### Tipos de objeto ( a nivel de BD )
- **tl**  = tablas
- **tr**  = triggers
- **pr**  = procedimientos
- **vw**  = views
- **fn**  = funciones

### Modulos
- **sto** = store
- **caj** = cash = caja
- **mov** = movimientos 
- **acc** = accounts = cuentas contables
- **usr** = users =  usuarios
- **cus** = customers
